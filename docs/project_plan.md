# Project Plan

This repository implements baseline pipelines for MRI-based neurological disease analysis.

Datasets:
- OASIS (Alzheimer classification)
- OpenNeuro Parkinson MRI datasets
- MSSEG 2016 (MS lesion segmentation)

Next stages:

1. Improve MS lesion segmentation
2. Add federated learning experiments
3. Extend interpretability methods
4. Refactor training code into reusable modules
